# E-Commerce-Analysis
Project ini mensimulasikan sistem database e-commerce yang terdiri dari data produk, toko, customer, transaksi, cart dan review.

# Ruang Lingkup Project

1) Project ini mencakup beberapa aspek utama, antara lain:
2)Perancangan struktur database menggunakan DDL
3) Pengolahan dan manipulasi data menggunakan DML
4) Analisis data melalui relasi antar tabel
5)Penerapan logika bisnis langsung pada level database

# Insight & Learning

1) Desain database yang baik mempermudah analisis dan menjaga konsistensi data
2) SQL berperan penting dalam menerapkan logika bisnis melalui query dan stored procedure
3) Relasi antar tabel menghasilkan insight yang lebih kontekstual
4) Database menjadi fondasi dalam pengambilan keputusan berbasis data

# Tools & Teknologi

1) MySQL
2) MySQL Workbench
3) SQL (DDL & DML)
